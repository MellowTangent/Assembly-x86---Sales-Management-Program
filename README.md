# Sales Management Program - Assembly x86

A menu-driven sales management program written in x86 Assembly using MASM and the Irvine32 library.

## Features
- Add sales entries stored in a dynamic array
- View all recorded sales
- View total sales amount
- Input validation and looping menu

## Requirements
- Microsoft Visual Studio with MASM support
- Irvine32 Library (available at asmirvine.com)

## How to Run
1. Clone the repo
2. Open the `.asm` file in Visual Studio
3. Link the Irvine32 library
4. Build and run with F5

## Concepts Demonstrated
- Procedure calls with register preservation using USES
- Pointer arithmetic for dynamic array traversal
- Conditional branching and loop control
- Console I/O using the Irvine32 library
- Register optimization (xor eax, eax vs mov eax, 0)
